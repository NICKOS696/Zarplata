# 🚀 НАЧНИТЕ ОТСЮДА!

## Добро пожаловать в систему учета продаж и расчета зарплаты!

Эта система решает вашу задачу: **автоматический расчет зарплаты с учетом планов, фактов продаж и посещаемости**.

---

## ⚡ Быстрый старт (3 шага)

### Шаг 1: Установка (5 минут)

Просто запустите файл:
```
install.bat
```

Он автоматически:
- Установит все зависимости
- Создаст базу данных
- Добавит тестовые данные

### Шаг 2: Запуск (1 минута)

**Откройте 2 терминала:**

**Терминал 1** (Backend):
```
start_backend.bat
```

**Терминал 2** (Frontend):
```
start_frontend.bat
```

### Шаг 3: Откройте браузер

Перейдите по адресу: **http://localhost:5173**

## 📚 Основные функции

### 1. Сотрудники
- Добавление/редактирование менеджеров и супервайзеров
- Установка фиксированной зарплаты
- Привязка к супервайзерам

### 2. Планы продаж
- Установка планов по сотрудникам
- Разбивка по брендам и KPI
- Гибкие периоды

### 3. Факты продаж
- Ручной ввод продаж
- Импорт из HTML файлов 1С
- Фильтрация и поиск

### 4. Табель посещаемости
- Учет рабочих дней
- Отметка присутствия/отсутствия
- Импорт из 1С

### 5. Расчет зарплаты ⭐
- **Автоматический расчет** по правилам
- Фиксированная часть (с учетом посещаемости)
- Мотивационная часть (% от продаж)
- Бонусы за перевыполнение
- Детализация расчета

### 6. Импорт из 1С
- Загрузка HTML файлов с продажами
- Загрузка HTML файлов с табелем
- Автоматическое сопоставление данных

### 7. Настройки
- Управление брендами
- Управление типами KPI
- Настройка правил расчета зарплаты

- Система автоматически рассчитает зарплату

## 🎓 Обучающие материалы

### Как импортировать данные из 1С?
1. Экспортируйте отчет из 1С в HTML
2. Перейдите в раздел "Импорт"
3. Выберите файл и нажмите "Импортировать"
4. Проверьте результат в логах

### Как настроить правила зарплаты?
1. Перейдите в "Настройки" → "Правила зарплаты"
2. Выберите тип правила:
   - **Процент от продаж** - базовый % + бонус за перевыполнение
   - **Ступенчатая система** - разные % для разных уровней выполнения
   - **Фиксированный бонус** - фиксированная сумма при достижении порога
3. Настройте параметры в JSON
4. Сохраните

### Как рассчитать зарплату?
1. Перейдите в "Зарплата"
2. Выберите период
3. Нажмите "Рассчитать всю команду"
4. Проверьте результаты в таблице
5. Посмотрите детали расчета

## ❓ Частые вопросы

### Q: Можно ли использовать PostgreSQL вместо SQLite?
**A:** Да! Просто измените `DATABASE_URL` в файле `backend/.env`

### Q: Как добавить нового сотрудника?
**A:** Перейдите в "Сотрудники" → "Добавить сотрудника"

### Q: Что делать, если изменилась структура команды?
**A:** Просто измените данные сотрудника или добавьте нового. Система автоматически адаптируется.

### Q: Как настроить сложную мотивационную программу?
**A:** Создайте несколько правил для одной должности. Они будут применяться одновременно.

### Q: Можно ли экспортировать отчеты?
**A:** Пока нет, но это можно добавить в будущем.

### Q: Где хранятся данные?
**A:** В файле `backend/sales.db` (SQLite база данных)

### Q: Как сделать бэкап?
**A:** Просто скопируйте файл `backend/sales.db`

## 🆘 Помощь

### Если что-то не работает:

1. **Backend не запускается:**
   - Проверьте, что установлен Python 3.9+
   - Запустите: `pip install -r backend/requirements.txt`
   - Проверьте файл `backend/.env`

2. **Frontend не запускается:**
   - Проверьте, что установлен Node.js 16+
   - Запустите: `cd frontend && npm install`
   - Убедитесь, что backend запущен

3. **Ошибка импорта из 1С:**
   - Проверьте формат HTML файла
   - Убедитесь, что сотрудники добавлены в систему
   - Имена в файле должны совпадать с именами в системе

4. **Неправильный расчет зарплаты:**
   - Проверьте правила расчета в настройках
   - Убедитесь, что есть планы продаж
   - Проверьте факты продаж за период
   - Проверьте табель посещаемости
